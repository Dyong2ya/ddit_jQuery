의사코드 : 상태나 구조에 의해서 선택 
   
   상태
   : link -- 처음 a링크가 생겼을 때
   : visited -- 한 번 방문했을 때
   : hover -- 마우스를 올려놨을 때
   : active -- 마우스를 누를 때
   구조
   : nth-child(2n) -- 2의 배수, 짝수번 째 자식을 선택
   : nth-child(3n) -- 3의 배수
   : nth-child(2n+1) -- 홀수
   : nth-child(even) -- 짝수
   : nth-child(odd) --홀수
   : nth-child(4) -- 4번째 자식을 선택
 
    a:link { color : red;}
    a:visited { color : green;}
    a:hover { background : pink;}
    a:active{ font-size : 1.2em; color : blue;}
