indexOf(item, start) : item의 위치를 검색 -->start에 위치를 넣어주는 거야
lastIndexOf(item, start) : item의 위치를 뒤에서부터 검색해서 위치를 return한다



push(item) : item을 배열의 마지막에 추가
pop() : 배열의 마지막 요소를 삭제



slice(start, end) : start ~ end 사이의 요소를 잘라서 새로운 배열을 만든다
end는 포함되지 않는다
splice(idx, n, a, b, c) : idx번째에서 n개를 삭제하고 a,b,c를 추가한다.
splice(idx, 0, a, b) : idx번째에서 삭제없이 a,b추가
splice(idx, n) : idx번째에서 추가되는 거 없이 n개 삭제만



concat : 여러개의 배열을 합친다
join : 한 개의 배열이 문자열로 변경한다.
