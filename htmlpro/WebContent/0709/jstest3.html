. script에서 html요소에 접근 검색
 	document.getElementsByTagName('tag이름')
	document.getElementsByClassName('class이름')
  	document.getElementById('id이름') -- id는 하나밖에 없어서 element 단수로 쓴다.
 
 . 검색된 html요소에 출력
 	innerHTML : 출력내용에 html태그가 포함. 없어도 무관함
 	textContent : 출력내용에 html태그가 포함되지 않을 경우
 	value : 입력하거나 선택의 경우 -> input type=text | input type=password | type=radio | type=checkbox | select-option | textarea
