백틱(backtick), backquote, grave accent를 이용하여 표현된 문자열을 
템플릿 리터럴이라고 한다

템플릿 리터럴내에서 변수값은 ${}에 표현된다
str = 이름 : "  + $('#user').val() + "
";
str = `이름 : ${$('#user').val()} 
` 

color="rgb(" + r + "," + g + "," + b + ")";
color = `rgb( ${r}, ${g}, ${b})`
